Flexbox Cheat Sheet üìù

Property Values What it does
display flex Creates flex container
flex-direction row, column, row-reverse, column-reverse Sets main axis direction
justify-content start, center, end, between, around, evenly Aligns along main axis
align-items start, center, end, stretch Aligns along cross axis
flex-wrap nowrap, wrap, wrap-reverse Controls wrapping
flex 1, 0 0 200px, etc. Item grow/shrink/basis
