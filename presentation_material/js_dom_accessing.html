3. Accessing the DOM

üî∏ Common DOM Access Methods

Method Description Returns
document.getElementById() Returns a single element with a specific ID Single Element
document.getElementsByClassName() Returns an HTMLCollection of elements HTMLCollection
document.getElementsByTagName() Returns elements with a given tag name HTMLCollection
document.querySelector() Returns the first element that matches a CSS selector Single Element
document.querySelectorAll() Returns a NodeList of elements matching a CSS selector NodeList

üìù Example Code:

const heading = document.getElementById('heading');
const paragraph = document.querySelector('.description');
const allButtons = document.querySelectorAll('button');

4. Manipulating the DOM

Now that we can select elements, let's learn how to change them!

üìù

üî∏ Change Content

Methods for Changing Content:

  • textContent - Changes only the text (safe from HTML)
  • innerHTML - Changes HTML content (can include tags)
heading.textContent = "Hello, Students!";
paragraph.innerHTML = "<strong>Let's learn JavaScript</strong>";
