What is CSS? ü§î

  • CSS stands for Cascading Style Sheets
  • It's a language used to style HTML elements
  • Controls layout, colors, fonts, spacing, and responsiveness
  • Makes websites look professional and attractive

Think of HTML as the skeleton, CSS as the skin and clothes!

Why Use CSS? üí°

  • Separation of concerns: Content (HTML) vs Style (CSS)
  • Consistency: Apply same styles across multiple pages
  • Maintainability: Change one file, update entire website
  • Flexibility: Create responsive designs for all devices
  • Performance: Faster loading and better user experience

CSS Syntax üìñ

selector { property: value; property: value; }

Example:

p { color: blue; font-size: 18px; font-weight: bold; }

This makes all paragraphs blue, 18px, and bold!

CSS Selectors üéØ

Element

p { color: red; }

Selects all <p> elements

Class

.btn { padding: 10px; }

Selects class="btn"

ID

#header { height: 100px; }

Selects id="header"

Universal

* { margin: 0; }

Selects all elements

CSS Box Model üì¶

Every HTML element is a rectangular box with:

Content
(Text/Images)
  • Content: The actual text or images
  • Padding: Space inside the border
  • Border: Line around the padding
  • Margin: Space outside the border

Common Units:

  • % - Percentage (relative to parent)
  • em - Relative to font size
  • rem - Relative to root font size
