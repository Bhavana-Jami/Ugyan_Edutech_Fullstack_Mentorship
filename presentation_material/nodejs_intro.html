What is Node.js?

üîç Key Concepts:

  • Runtime Environment: JavaScript outside the browser
  • V8 Engine: Same engine that powers Chrome
  • Event-Driven: Non-blocking, asynchronous I/O
  • Single-Threaded: But handles concurrency efficiently
  • NPM: Largest package ecosystem in the world

Core Modules (Built-in)

Node.js comes with powerful built-in modules. No installation required!

http
Built-in
Create HTTP servers and clients
const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello World!');
});

server.listen(3000);
fs
Built-in
File system operations
const fs = require('fs');

// Read file
fs.readFile('data.txt', 'utf8', (err, data) => {
  if (err) throw err;
  console.log(data);
});

// Write file
fs.writeFile('output.txt', 'Hello!', (err) => {
  if (err) throw err;
  console.log('File saved!');
});
path
Built-in
Work with file and directory paths
const path = require('path');

console.log(path.join('/users', 'john', 'documents'));
// Output: /users/john/documents

console.log(path.extname('file.txt'));
// Output: .txt
os
Built-in
Operating system utilities
const os = require('os');

console.log('Platform:', os.platform());
console.log('CPU Architecture:', os.arch());
console.log('Free Memory:', os.freemem());
console.log('Total Memory:', os.totalmem());

Database Integration

üìä Database Drivers & ORMs

mongoose
Latest: 8.0.3
MongoDB object modeling for Node.js
npm install mongoose
const mongoose = require('mongoose');

// Connect to MongoDB
mongoose.connect('mongodb://localhost:27017/myapp');

// Define schema

  name: String,
  email: String,
  age: Number
});

const User = mongoose.model('User', userSchema);
sequelize
Latest: 6.35.2
Promise-based ORM for PostgreSQL, MySQL, MariaDB, SQLite

const { Sequelize, DataTypes } = require('sequelize');

const sequelize = new Sequelize('database', 'username', 'password', {
  host: 'localhost',
  dialect: 'mysql'
});

const User = sequelize.define('User', {
  name: DataTypes.STRING,
  email: DataTypes.STRING
});
